Clearcode numeral filter
=========

Filter that returns number parsed in set format.

Usage
------

Add ``` cc.numeral.filter ``` module to your app module list :


```
angular
    .module('yourAwesomeApp', [
        'cc.numeral.filter'
    ]);
```
and you are ready to go!

Filters :

*numeral*

```
numeral(input, format);

\\ OR in html

numeral:format

```
